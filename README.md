# 水果新鮮度診斷 App

一個使用 Python Kivy 開發的 Android 應用程式，用於診斷水果的新鮮度和成熟度。

## 功能
- 📸 拍照分析功能
- 🖼️ 選擇圖片功能
- 📊 新鮮度評分
- 💡 食用建議

## 技術
- Python 3.9
- Kivy 框架
- Buildozer 打包

## 如何建置
使用 GitHub Actions 自動建置 APK。

## 如何安裝
1. 前往 Releases 頁面下載最新 APK
2. 在 Android 裝置上允許安裝未知來源的應用程式
3. 安裝並執行
